angular 表达式

一 支持部分

表达式 : 3 + 2 = {{ 3 + 2 }}

变量定义提升 : {{ a }} 另一个双大括号中还是 {{ a = 1 }}

三目表达式 : b = {{ b = true; b ? 3 : 2 }}

== 和 === : {{ 2 == '2' }} , {{ 2 === '2' }}

转换为大写 toUpperCase : angular -> {{ 'angular'.toUpperCase() }}

二 不支持部分

++和-- : {{z = 1; ++z}} {{z = 1; --z}}

for 循环 : {{for (y = 0; y < 4; y = y + 1) {y}}}

全局函数和对象 : {{Math.random()}}
